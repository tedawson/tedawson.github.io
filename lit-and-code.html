Literature, Culture, and Code

A flexible course companion for teaching humanists to code with Python. The first section teaches basic principles (Python syntax, variables, functions, branching, modularity) to the absolute novice by leading the learner through writing scripts to conduct basic analyses of texts stored online. The second section presents engaging projects for exploring humanities themes with Python. These projects are meant both to rigorously develop programming skills for humanists and to present humanistic applications to those from a computer science background. Projects explore such areas as computer-generated poetry and creative sound experimentation and data sonification. The final section provides a code based exploration of literary theories and a very brief introduction to critical code studies.

Ultimately envisioned as a web-based book, Literature, Culture, and Code is intended to offer a course for individual study, as well as a basic text that could be used in a semester-long course or to provide exercises for shorter workshops.

For an example of a semester course in which Literature, Culture, and Code is integrated, see my syllabus for a course called The 'Paragon of Animals' in a World of Computers: An Introduction to Humanistic Inquiry in the Digital Age.
