PowerShell Tutorial

My Introduction to PowerShell for The Programming Historian introduces humanists to working on the command line to do basic tasks and get started with important command-line tools such as Wget, Mallet, or Python scripting. While several command-line tutorials oriented towards humanists currently exist, including one on The Programming Historian, all teach Bash, and either expect Windows users to install Bash or include nods towards PowerShell, but treat it as a Windows version of Bash, and thus do not explain how to do many of the most important things, and sometimes even explain things incorrectly. This tutorial corrects that problem by being oriented specifically towards PowerShell. As such, it also hopes to correct an inadvertent bias in the Digital Humanities toward Mac users.
